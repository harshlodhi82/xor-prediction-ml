# xor-prediction-ml
This is XOR prediction ML ptoject, 
This program learns itself to predict the answer of XOR boolean operator
<br>
like:
<br>
True-True = False
<br>
False-False = False
<br>
True-False = True
<br>
False-True = True
<br>
<br>
Note: This is a learning project not a complete and optimized project.
